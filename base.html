# 接入基础脚本

# 原理

你可以通过手动在全局插入一段脚本，注入一个IIFE（立即执行函数），目的是动态添加一个 <script> 标签在当前脚本下方，用以并行加载行为收集的脚本。这种动态添加 <script> 标签的方式并不会阻塞js引擎对于现有项目的读取。例如：

(function () {
    var ha = document.createElement('script');
    ha.type = 'text/javascript';
    ha.async = true;
    // 脚本地址以真实地址为准
    ha.src = 'https://sales-int.ihxlife.com/video/ha-analytics/collect.js';
    var s = document.getElementsByTagName('script')[0];
    s.parentNode.insertBefore(ha, s);
})();

另外需要在全局手动初始化一个数组，变量名称请统一设置为_haq（重要），用于后续作为命令队列使用。例如：

var _haq = _haq || [];

# 插入位置

应该在 </head> 标签之前插入该脚本。

# 示例代码

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>title</title>
    <link rel="stylesheet" href="./reset.css">
    <!-- HuaXia Analytics -->
    <script>
        var _haq = _haq || [];
        (function () {
            var ha = document.createElement('script');
            ha.type = 'text/javascript';
            ha.async = true;
            // 脚本地址以真实地址为准
            ha.src = 'https://sales-int.ihxlife.com/video/ha-analytics/collect.js';
            var s = document.getElementsByTagName('script')[0];
            s.parentNode.insertBefore(ha, s);
        })();
    </script>
    <!-- End HuaXia Analytics -->
</head>
<body>
    
</body>
</html>

注意：

如果现有项目是多页面架构的，则需要在所有分页头部加入此段基础脚本，若是单页面应用（类似基于Vue构建的项目），则只需要在唯一的模板头部加入此段基础脚本。

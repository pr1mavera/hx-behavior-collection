# 介绍

此文档用于帮助官微前端开发人员更好的配置和收集用户的行为数据。

# 准备工作

在正式接入之前，你需要收集以下数据，在注册授权信息时作为一部分配置信息传入，具体数据如下：

key 名称 格式 示例
sysId 应用Id String " HX-XXXX-X "
sysName 应用名称 String " 华夏人寿客服系统 "
openId 客户openId String " 98qwyu34nnduiafghs9dkdfgjnho "
origin 客户来源 String " WE "
timestamp 页面进入时记录的时间戳 String " 1554958354428 "

说明：

  • sysId 和 sysName，可以通过登录华夏数据分析后台注册应用获取
  • openId 和 origin，需要开发人员从官微渠道获取到用户信息。
  • timestamp 客户端进入时间

# 命令 - 页面载入

通过关键字 _enterPage 可以触发页面载入命令。

# 参数

你可以以数组的方式顺序传入，需按照给定顺序：

_haq.push([ '_enterPage', title, pageUrl, desc, timestamp ])
  • title (required)
    当前进入页面的名称（可以是 <head> 里面的 <title> ）
  • pageUrl (required)
    当前页面路由
  • desc (required)
    当前页面描述
  • timestamp (optional)
    页面载入时记录的时间戳

你也可以包装成对象整体传入：

_haq.push(['_enterPage', {
    title: '百度首页',
    pageUrl: 'http://www.baidu.com/',
    desc: '进入百度搜索首页',
    timestamp: `${timestamp}`
}])

# 调用时机

你可以在当前项目进入页面的时候传入页面载入的命令，用于记录用户载入某个页面的行为。

// SPA应用(Vue)
mounted() {
    this.$nextTick(() => {
        _haq.push(['_enterPage', ...restOptions])
    })
}

// 多页
window.onload = () => {
    _haq.push(['_enterPage', ...restOptions])
}

注意：

这里的进入页面指的是所有的触发形式，包括初始化时页面载入、单页面应用的路由跳转以及用户触发的回退操作，都需要传入该命令。如果有遗漏部分，则有可能导致页面停留时长等信息丢失！

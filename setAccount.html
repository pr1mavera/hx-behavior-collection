# 命令 - 注册授权

通过关键字 _setAccount 可以触发注册授权命令。

注意：

  • 注册授权命令是作为初始化行为收集脚本的首个配置类命令，你需要确保先传入该命令进行初始化配置，再尝试传入其他命令。
  • 单个分页只需要初始化配置一次，如果多次配置则会忽略后续命令。

# 参数

你可以以数组的方式顺序传入，需按照给定顺序：

_haq.push([ '_setAccount', debug, sysId, sysName, timestamp, openId, origin ])
  • debug (required)
    true：开启调试模式，调用的所有api的返回值会在客户端alert出来
    false：关闭调试模式，调用的所有api的返回值会在客户端控制台打印
  • sysId (required)
    系统的唯一标识
  • sysName (required)
    系统名称
  • timestamp (required)
    页面进入时记录的时间戳
  • openId (required)
    客户openId
  • origin (required)
    客户来源

你也可以包装成对象整体传入：

_haq.push(['_setAccount', {
    debug: `${true}`,
    sysId: `${sysId}`,
    sysName: `${sysName}`,
    timestamp: `${timestamp}`,
    openId: `${openId}`,
    origin: `${origin}`
}])

# 调用时机

  • 你可以在程序的任意位置任意时机调用
  • 注册完成之后，便可以使用其他命令
